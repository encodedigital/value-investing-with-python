# value-investing-with-python
This is repository for series "Benjamin Graham Value Investing Analysis with Python" on Youtube
Link to the complete youtube series is https://www.youtube.com/playlist?list=PLaV0Lkv57OLcIWFNVBd2lszSZbRaUMVjI

In this series, we sourced, saved, manipulated and analysed last 10 years of financial data in PostgreSQL of roughly around 3000-4000 US Stocks. 
Then based on criterias mentioned by Benjamin Graham in his famous book "The Intelligent Investor", we then filter those codes by our Python algorithm.

Check the complete series on Youtube.
